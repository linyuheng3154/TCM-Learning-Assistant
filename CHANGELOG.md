# 更新日志

## [0.4.1] - 2026-03-19

### 重大更新
- 🎉 **前端界面 MVP** - 完整的单页应用，支持方剂库、药材库浏览与搜索
- 📖 **经典原文数据** - 31首方剂的经典原文及出处，25首方剂的煎服法原文
- 🔧 **炮制信息** - 689条药材炮制信息（如"炙"、"擘"、"去心"等）

### 新增功能
- 🖥️ **前端界面** - 深色主题响应式界面，无框架依赖
- 📋 **方剂详情弹窗** - 展示组成、用法、原文、方剂变化等完整信息
- 🌿 **药材详情弹窗** - 点击方剂组成中的药材即可查看详情
- 🔍 **拼音搜索优化** - 支持汉字级拼音首字母精确搜索
- 📝 **数据模型扩展** - 新增 classic_text、classic_source、classic_usage 字段

### Bug 修复
- 🐛 修复方剂变化列表显示 undefined 的问题
- 🐛 修复药材搜索 limit 超限导致 422 错误
- 🐛 修复页面刷新时 JS 空指针错误
- 🐛 优化不存在的药材点击提示

### 技术改进
- ⚡ API 搜索限制调整为 50 条
- 🔒 智能辨证模块移至 .gitignore，待验证后发布
- 📁 新增 /ui 路由提供前端静态文件服务
