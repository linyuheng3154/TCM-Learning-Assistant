# TCM-Learning-Assistant

## 项目描述
中医学习助手 - 结合传统中医知识与现代技术的开源项目

## 核心功能
- 方剂智能查询系统 ✅
- 药材知识管理系统 ✅
- AI辅助辨证分析 🔶 开发中

## 技术栈
- **后端**: FastAPI + Python
- **前端**: React + TypeScript (规划中)
- **AI集成**: 自然语言处理技术
- **数据存储**: JSON文件 → SQLite

## 快速开始
```bash
# 安装依赖
pip install -r requirements.txt

# 启动开发服务器
uvicorn src.main:app --reload
```

## 开发状态
🚧 积极开发中 - 探索中医知识数字化路径

## 项目目标
- 构建实用的中医学习辅助工具
- 推动传统医学的现代化传承
- 为开源社区贡献中医数字化经验
